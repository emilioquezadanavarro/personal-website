# Emilio Quezada - Personal Website

Welcome to the repository for my personal website, live at [www.emilioquezada.com](https://www.emilioquezada.com).

This is a fast, lightweight, and minimalistic portfolio built to showcase my work as a AI Engineer.

## 🚀 Tech Stack
This project purposely avoids heavy frontend frameworks and relies on native, clean web technologies for maximum performance and rapid loading:
- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Custom design system featuring a structured "Highlighter" aesthetic.
- **Vanilla JavaScript**: Lightweight logic handling the interactive Dark/Light mode theme toggle.

## ✨ Features
- **Custom Theme Engine**: Includes a Dark Mode and Light Mode, instantly swapping colors, backgrounds, and profile pictures based on the active theme.
- **Highlighter Interface**: A unique, color-matched neon yellow (`#f5f258`) highlighter effect for text emphasis and button states.
- **Fully Responsive**: Beautifully adapts from widescreen monitors down to strictly mobile devices.
- **Zero Dependencies**: Pure code—no NPM packages, no build steps, and no compilation required.

## 🛠️ Local Development
Because this is a static site, you don't need a development server to view it:
1. Clone the repository to your local machine.
2. Double-click `index.html` to open it directly in your favorite web browser.
3. Any edits made to `styles.css` or `index.html` will be immediately visible upon refreshing the browser.
